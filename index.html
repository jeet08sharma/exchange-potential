Overview

Meta-GGA functionals depend on the kinetic energy density τ in addition to the density and its gradient. The semilocal exchange potential is the functional derivative of the exchange energy with respect to the electron density, evaluated in the semilocal approximation:

vxsl(r) = εxLDA Fx(s, ᾱ) + n · ∂[εxLDA Fx]/∂n − ∇ · { n · ∂[εxLDA Fx]/∂(∇n) }

Radial derivatives are computed using the Lagrange derivative method (Seino & Nakai) on the non-uniform Koga radial grid. The iso-orbital indicator ᾱ — central to SCAN and r²SCAN — is regularized differently in each functional variant studied here.

Functionals & Variants

SCAN — original 2015 functional
r²SCAN — regularized-restored SCAN
r²SCAN@LKT — LKT τ denominator
r²SCAN@PGS — PGS τ denominator
Variant τ denominator (ᾱ definition) Motivation
SCAN τ_TF + η·τ_W Satisfies all 17 known exact constraints
r²SCAN τ_TF·exp(−40p/27) + τ_W (PGS) Removes numerical difficulties of SCAN
r²SCAN@LKT τ_TF·sech(1.3·s) + τ_W LKT iso-orbital indicator regularization
r²SCAN@PGS τ_TF·exp(−40p/27) + τ_W PGS regularization of reduced gradient term

Repository Structure

All .dat files are plain-text, two-column (r in Bohr, vxsl in Hartree).

Reproducing the Results

# Clone the repository
git clone https://github.com/YOUR_USERNAME/exchange-potential.git
cd exchange-potential

# Install dependencies
pip install numpy matplotlib

# Example: Hydrogen
cd H/R2SCAN  &&  python vx_r2scan.py   # → r2scan.dat, r2scan_LKT.dat, r2scan_PGS.dat
cd ../SCAN   &&  python vx_scan.py      # → scan.dat
cd ..        &&  python plot.py          # → H_vx_r2scan_scan.pdf
